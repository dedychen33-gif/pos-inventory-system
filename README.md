# POS & Inventory Management System

Sistem manajemen stok dan point of sale (POS) yang lengkap dengan fitur:

## Fitur Utama
- ✅ Dashboard real-time dengan analytics
- ✅ Point of Sale (POS) dengan multi payment
- ✅ Manajemen Produk & Varian
- ✅ Manajemen Stok Multi-Gudang
- ✅ Pembelian & Purchase Order
- ✅ Customer & Loyalty Program
- ✅ Laporan & Analytics
- ✅ User Management & Role-based Access

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build
```
